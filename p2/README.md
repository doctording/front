
# border（边框）

## border-width

* 不支持百分比（由使用场景和语义决定）

* 支持关键字

    1. thin 1px

    2. medium 3px

    3. thick 5px

## border-style

* solid（实线）

* dashed （虚线）

* dotted (点线)

* double (双线)

* inset (内凹)

## border-color

* 不指定，其默认颜色是color

* transparent 透明的

## border & backgroud

```css
backgroud-position: 50px; /*相对左边*/
```
